plumbing Class

Defined in: c:\projects\flocks.js\lib\flocks.jsx:695
Module: flocks

plumbing is the flocks mixin which handles all of the behind-the-scenes work to make flocks function. The mixin is automatically added by calling flocks.createClass, or you can add it yourself in the standard mixin fashion if you prefer.
